Introduction to UnRTF

UnRTF is a command-line program written in C which converts documents in Rich Text Format (.rtf) to HTML, LaTeX, PostScript, troff macros, and RTF itself. Converting to HTML, it supports a number of features of Rich Text Format:

  • Changes in the text's font, size, weight (bold), and slant (italic)
  • Underlines and strikethroughs
  • Partial support for text shadowing, outlining, embossing, or engraving
  • Capitalizations
  • Superscripts and subscripts
  • Expanded and condensed text
  • Changes in the foreground and background colors
  • Conversion of special characters to HTML entities
Recent development has concentrated on conversion to HTML, as it can in turn be converted to other formats. Support for features when converting to other file formats using unrtf varies.

Commencing with version 0.21.0 all codepage conversion is done through iconv, and all output control is through runtime configuration files, including for particular font encodings. Thus the output of unrtf can be easily configured for new formats, and the output for existing formats can be tuned.

Downloading UnRTF
